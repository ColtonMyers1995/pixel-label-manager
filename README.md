# Pixel Label Manager (ArkOS)

A controller-friendly tool for creating custom system/category labels for the PIXEL OS theme.

## Features
- Icon selection (default + custom packs)
- Grid-based text input (controller friendly)
- Multiple fonts
- Multiple colors
- Automatic PNG generation

## Installation
Copy the script to:
```
/roms/tools/
```

Make executable if needed:
```
chmod +x PixelLabelManager.sh
```

## Usage
- Launch from Tools in ArkOS
- Create your label
- Save to art, art2, or art3
